AUTÓVERSENYZŐK LISTÁJA

Készítette: Szorcsik István - EPNXJJ
Az egyes API végpontok működése és elérése:
1.) api/drivers: JSON formátumban megjeleníti a hozzáadott versenyzők adatainak kulcs-érték párjait
2.) api/drivers/count: Megjeleníti a hozzáadott versenyzők számát
3.) api/drivers/id: JSON formátumban megjeleníti az egyes versenyzők adatait, azonosítók szerint külön-külön (pl.: api/drivers/2)
